Introduction

Dart is a modern, object-oriented programming language designed for building scalable, high-performance web and mobile applications. This documentation provides an overview of key Dart concepts and features.

Installation

To begin programming in Dart, follow these steps for installation:

  1. Visit the Dart SDK website.
  2. Download the Dart SDK based on your operating system (Windows, macOS, or Linux).
  3. Follow the installation instructions provided for your specific platform.

IDEs and Editors

Enhance your Dart development experience with integrated development environments (IDEs) or editors:

  • Visual Studio Code: Install the Dart and Flutter extensions for a feature-rich experience.
  • Android Studio: Suitable for Flutter development with Dart support out of the box.
  • IntelliJ IDEA: Offers a Dart plugin for comprehensive Dart development.

Using DartPad

Experiment with Dart in a web-based environment using DartPad. No installation is needed; simply visit the website, write your Dart code, and run it instantly.

Dart Packages

Explore and use Dart packages to extend functionality. The Dart Package Repository (pub.dev) is a rich source of Dart packages. Use the pub command-line tool to manage dependencies.

Hello World

Once Dart is installed, create a simple "Hello, World!" program to verify your setup:


void main() {
	print('Hello, World!');
}

Save this file with a .dart extension (e.g., hello.dart). Open a terminal, navigate to the file's directory, and run:

dart hello.dart

The program prints Hello, World! to the console.

Variables

A variable is a location in memory (storage area) to hold data. To indicate the storage area, each variable should be given a unique name (identifier). Variable names are just the symbolic representation of a memory location.

Declaring Variables

To declare a variable, use the built-in keyword var:

var name = 'Bob';

Here, name is a variable of type String. The type of variable is inferred at compile time if not explicitly specified.

Alternatively, you can explicitly declare the type:

String name = 'Bob';

Variable scope

The scope of a variable determines its visibility. Dart supports lexical scope. Scopes can be nested. Labels are the only exception to the lexical scope rule.

Dart supports the following types of variable scope:

  • Class-level variables: Variables declared outside methods, constructors or blocks are class-level variables. These variables are visible throughout the class.
  • Local variables: Variables declared within methods, constructors or blocks are local variables.
  • Formal parameters: Variables declared as formal parameters for methods, constructors or lambda expressions are formal parameters. 

	void exampleFunction() {
  String localVariable = 'Local';
  print(localVariable);
}

// Uncommenting the next line would result in an error since localVariable is not accessible here.
// print(localVariable);

Constants

Constants are variables whose values cannot be changed. Dart supports two types of constants:


const pi = 3.14;

For more information, see Variables in the Dart Language Tour.

Data Types

Dart is a statically typed language, which means that variables always have a specific type associated with them. However, type annotations are optional because Dart can infer types. In addition, Dart supports generic types, enabling the creation of classes and collections that are type-safe at compile time.

Numbers

Dart numbers come in two flavors: int and double. Dart numbers are automatically converted from one type to another as necessary.


int age = 10;
double price = 5.99;

Strings

Dart strings are sequences of UTF-16 code units. Strings are immutable.


String name = 'Bob';
String message = 'Hello, $name!';

Booleans

Dart booleans represent true and false values and are used for conditional tests.


bool isRaining = true;
bool isSunny = false;

Lists

Dart lists are ordered collections of objects. Lists can contain objects of different types.


var list = [1, 2, 3];
print(list[0]); // 1

if...else Statements

Use if statements to execute code only if a condition is true:


if (isRaining) {
	print('Bring an umbrella.');
}

Use if...else statements to execute code when a condition is true, or to execute different code if the condition is false:


if (isRaining) {
	print('Bring an umbrella.');
} else {
	print('Wear sunscreen.');
}

Use if...else if...else statements to specify a new condition to test, if the first condition is false:


if (isRaining) {
	print('Bring an umbrella.');
} else if (isSnowing) {
	print('Wear a coat.');
} else {
	print('Wear sunscreen.');
}

For more information, see if and else in the Dart Language Tour.

while and do...while Loops

Execute code repeatedly as long as a condition is true using while loops: 


while (isRaining) {
	print('Keep your umbrella open.');
}

Execute code repeatedly until a condition becomes true using do...while loops:


do {
	print('Keep your umbrella open.');
} while (isRaining);

For more information, see while and do-while in the Dart Language Tour.

Functions

Functions are reusable blocks of code that perform a task. Dart supports the following types of functions:

  • Named functions: Functions that are explicitly named.
  • Anonymous functions: Functions that are not explicitly named. Also known as lambda expressions.
  • Methods: Functions that are defined inside classes.

Named Functions

Use named functions to perform a task. For example, the following function prints a message to the console:


void printMessage() {
	print('Hello!');
}

Call the function to execute the code inside it:


printMessage();

Functions can take parameters:


void printMessage(String message) {
	print(message);
}

Call the function and pass in an argument:


printMessage('Hello!');

Functions can return a value:


int sum(int a, int b) {
	return a + b;
}

Call the function and store the return value:


int result = sum(1, 2);
print(result); // 3

Anonymous Functions

Use anonymous functions to pass a function as an argument to another function. For example, the following function takes a function as an argument and calls that function:


void printMessage(Function messageFunction) {
	messageFunction();
}

Call the function and pass in an anonymous function as an argument:


printMessage(() {
	print('Hello!');
});

Lists and Collections

Dart lists are ordered collections of objects. Lists can contain objects of different types. Dart supports the following types of lists:

  • Fixed-length lists: Lists with a fixed length.
  • Growable lists: Lists with a variable length.

Fixed-length Lists

Use a fixed-length list when you know the number of elements that the list will contain. For example, the following code creates a fixed-length list with a length of two:


var list = new List(2);
list[0] = 'a';
list[1] = 'b';
print(list); // [a, b]

Attempting to add more elements to the list results in an error:


list[2] = 'c'; // Error: RangeError (index): Index out of range: index should be less than 2

Growable Lists

Use a growable list when you don't know the number of elements that the list will contain. For example, the following code creates a growable list:


var list = ['a'];
list.add('b');
print(list); // [a, b]

For more information, see Lists in the Dart Language Tour.

Maps and Sets

Dart provides specialized syntax for creating and initializing maps and sets.

Maps

In general, a map is an object that associates keys and values. Both keys and values can be any type of object. Each key occurs only once, but you can use the same value multiple times.

Here are a couple of simple Dart maps, created using map literals:


var gifts = {
  // Key:    Value
  'first': 'partridge',
  'second': 'turtledoves',
  'fifth': 'golden rings'
};

var nobleGases = {
  2: 'helium',
  10: 'neon',
  18: 'argon',
};

Sets

A set in Dart is an unordered collection of unique items. Dart support for sets is provided by set literals and the Set type.

Here is a simple Dart set, created using a set literal:


	var halogens = {'fluorine', 'chlorine', 'bromine', 'iodine', 'astatine'};

For more information, see Maps and Sets in the Dart Language Tour.

Classes and Objects

Dart is an object-oriented language with classes and mixin-based inheritance. Every object is an instance of a class, and all classes except Null descend from Object. 


	class Person {
  String name;
  int age;

  Person(this.name, this.age);

  void celebrateBirthday() {
    age++;
    print('$name is now $age years old!');
  }
}

void main() {
  Person john = Person('John', 25);
  john.celebrateBirthday();
}

Exception Handling

Dart provides try, catch, and finally blocks for handling exceptions. 


void divide(int a, int b) {
  try {

    print(a ~/ b);  // Integer division

  } catch (e) {

    print('Error: $e');

  } finally {

    print('Division operation complete.');

  }
}

void main() {
  divide(10, 2);
  divide(5, 0);  // This will trigger the catch block.
}

For more information, see Exceptions in the Dart Language Tour.

Asynchronous Programming

Dart supports asynchronous programming using Future and async/await.

Futures

A Future object represents a potential value or error that will be available at some time in the future. Receiving its value is known as resolving the Future. 


Future<void> fetchData() async {
  print('Fetching data...');
  await Future.delayed(Duration(seconds: 2));
  print('Data fetched!');
}

async/await

Use the async and await keywords to write asynchronous code that looks similar to synchronous code. 


Future<void> fetchData() async {
	print('Fetching data...');
	await Future.delayed(Duration(seconds: 2));
	print('Data fetched!');
}

Libraries and Packages

A Dart library is a collection of code. Libraries can be reused and imported into other libraries. Dart libraries begin with the library keyword, followed by the library name and a semicolon. 


import 'dart:math';

void main() {
  print('Random number: ${Random().nextInt(10)}');
}
